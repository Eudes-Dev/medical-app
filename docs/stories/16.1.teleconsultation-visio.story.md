# Story 16.1: Téléconsultation (visio sécurisée)

## Status
Draft

## Story
**As a** patient,
**I want** réaliser une consultation à distance via vidéo,
**so that** je n'ai pas à me déplacer pour un suivi simple.

## Acceptance Criteria
1. Un `ServiceType` peut être flagué `isTeleconsultation`.
2. À l'heure du RDV, un lien sécurisé (token) est actif 15 min avant et 30 min après.
3. Visio peer-to-peer (WebRTC) avec fallback TURN ; pas d'enregistrement.
4. Salle d'attente virtuelle ; le patient entre quand le praticien accepte.
5. Conformité données de santé : hébergement HDS pour signaling.
