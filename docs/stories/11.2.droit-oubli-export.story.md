# Story 11.2: Droit à l'oubli et portabilité

## Status
Draft

## Story
**As a** patient,
**I want** demander l'export ou la suppression de mes données personnelles,
**so that** mes droits RGPD (articles 17 et 20) sont effectifs.

## Acceptance Criteria
1. Formulaire `/legal/data-request` (lien signé envoyé après vérification email).
2. Export : ZIP contenant JSON des données + documents associés.
3. Suppression : anonymisation des RDV passés (conservation comptable) + purge des champs identifiants.
4. Délai légal de 1 mois respecté ; statut visible côté praticien dans `/dashboard/data-requests`.

## Tasks / Subtasks
- [ ] Server Action `requestDataExport` / `requestDataDeletion`
- [ ] Worker asynchrone (Vercel Cron + Supabase queue)
- [ ] Tampon de rétention légale (10 ans pour les dossiers médicaux FR)
- [ ] Anonymisation : firstName="ANON", phone=null, email=null, notes=null
