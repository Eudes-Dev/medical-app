# Story 7.3: Types de soins paramétrables

## Status
Draft

## Story
**As a** praticien,
**I want** configurer mes types de consultations (nom, durée, couleur, tarif, description publique),
**so that** les patients choisissent un motif adapté et que mon agenda reflète la durée réelle.

## Acceptance Criteria
1. CRUD complet des types de soins via `/dashboard/settings/services`.
2. Chaque type a : libellé, durée (min), couleur (affichage calendrier), tarif optionnel, description publique, actif/inactif, public/privé.
3. Les types publics apparaissent dans le tunnel `/book` ; les privés sont réservés à la création depuis le dashboard.
4. La couleur est appliquée aux blocs de RDV dans le calendrier.
5. Suppression bloquée si des RDV existants y sont rattachés (proposer désactivation).

- [ ] Modèle Prisma `ServiceType` (label, durationMin, color, price?, description, isPublic, active)
- [ ] Migration : remplacer `Appointment.type: String` par `serviceTypeId: String` (avec fallback string preservé)
- [ ] CRUD UI Shadcn (table + drawer)
- [ ] Sélecteur de couleur (palette restreinte de 8 couleurs)
- [ ] Refactor tunnel `/book` step 1 pour afficher les services publics
