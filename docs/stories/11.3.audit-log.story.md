# Story 11.3: Audit log

## Status
Draft

## Story
**As a** praticien (et auditeur potentiel),
**I want** un journal d'audit immuable des actions sur les données patients,
**so that** je peux tracer toute modification en cas de litige.

## Acceptance Criteria
1. Toute création/lecture/modification/suppression d'un patient, RDV, document est journalisée (qui, quand, quoi, IP, userAgent).
2. Les entrées d'audit sont en append-only (pas d'UPDATE/DELETE possible).
3. Consultation depuis `/dashboard/audit` avec filtres (date, entité, action).
4. Conservation 5 ans minimum.

## Tasks / Subtasks
- [ ] Modèle Prisma `AuditLog` (actorId, action, entity, entityId, before JSONB, after JSONB, ip, ua)
- [ ] Wrapper Server Action `withAudit(action)`
