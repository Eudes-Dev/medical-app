# Story 18.2: Import patients depuis Doctolib / CSV

## Status
Draft (Future)

## Story
**As a** praticien migrant d'un autre logiciel,
**I want** importer ma base patients existante en CSV ou export Doctolib,
**so that** je ne repars pas de zéro.

## Acceptance Criteria
1. Page `/dashboard/settings/import` avec upload CSV.
2. Mapping des colonnes interactif.
3. Détection des doublons (email/téléphone) avec choix de fusion.
4. Aperçu avant import + rapport post-import (X créés, Y fusionnés, Z erreurs).
5. Rollback possible sur les 24h suivantes (soft delete).
