# Story 12.1: Acompte en ligne via Stripe

## Status
Draft

## Story
**As a** praticien,
**I want** demander un acompte non remboursable à la réservation pour certains types de soins,
**so that** je réduis les no-shows.

## Acceptance Criteria
1. Sur les `ServiceType` flaggés `requiresDeposit`, le tunnel `/book` ouvre Stripe Checkout avant confirmation.
2. RDV créé en statut `PENDING_PAYMENT` jusqu'au webhook de succès → `CONFIRMED`.
3. En cas d'abandon, le RDV est purgé après 15 minutes.
4. Le montant de l'acompte est configurable par service.
