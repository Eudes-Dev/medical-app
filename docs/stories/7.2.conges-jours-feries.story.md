# Story 7.2: Congés et jours fériés

## Status
Draft

## Story
**As a** praticien,
**I want** bloquer des journées ou des plages spécifiques (congés, formation, jours fériés),
**so that** aucun RDV ne peut être pris sur ces périodes.

## Acceptance Criteria
1. Le praticien peut créer des "exceptions" (date unique ou plage de dates) depuis `/dashboard/settings/timeoff`.
2. Les jours fériés français sont pré-remplis annuellement et désactivables.
3. Les exceptions masquent les créneaux dans le tunnel `/book` et dans le calendrier.
4. Une exception qui chevaucherait des RDV existants déclenche un avertissement et propose une notification d'annulation aux patients concernés.

## Tasks / Subtasks
- [ ] Modèle Prisma `TimeOff` (startDate, endDate, reason, allDay)
- [ ] Service `holidays.ts` (API ou liste statique des jours fériés FR)
- [ ] UI : calendrier sélection plage (Shadcn DateRangePicker)
- [ ] Intégration dans le générateur de créneaux
- [ ] Job d'avertissement RDV impactés
