# Story 18.3: Synchronisation Google / Apple Calendar

## Status
Draft (Future)

## Story
**As a** praticien,
**I want** synchroniser mon agenda du cabinet avec Google Calendar (lecture + écriture),
**so that** mes RDV personnels et professionnels cohabitent sans conflit.

## Acceptance Criteria
1. OAuth Google avec scope Calendar.
2. Synchronisation bidirectionnelle des RDV (création / update / delete).
3. Champs sensibles non syncs (notes médicales restent dans l'app).
4. Conflits affichés dans le calendrier (événements externes en gris).
5. Désynchronisation propre.

- [ ] Google Calendar API + OAuth
- [ ] Webhooks Google (push notifications)
- [ ] Mapping appointment ↔ event
- [ ] Tests deduplication
