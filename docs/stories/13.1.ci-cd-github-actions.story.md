# Story 13.1: Pipeline CI/CD GitHub Actions

## Status
Draft

## Story
**As a** développeur,
**I want** un pipeline CI qui exécute lint, typecheck, tests et build à chaque PR,
**so that** la qualité est garantie avant merge.

## Acceptance Criteria
1. Workflow `.github/workflows/ci.yml` : install (cache), lint, typecheck, vitest, build Next.js.
2. Matrice Node 20 / 22.
3. Statut requis sur PR avant merge (branch protection main).
4. Cache npm + cache Next.
5. Préview Vercel déclenchée automatiquement sur chaque PR.
