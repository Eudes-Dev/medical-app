# Story 6.2: Rappels automatiques de rendez-vous

## Status
Draft

## Story
**As a** praticien,
**I want** que mes patients reçoivent un rappel automatique avant leur RDV,
**so that** je réduis le taux de no-show.

## Acceptance Criteria
1. Un rappel email est envoyé J-1 à 18h00 (heure cabinet) pour tout RDV CONFIRMED du lendemain.
2. Un rappel "H-2" est envoyé 2h avant le RDV.
3. Le système est idempotent : pas de doublon même si le job est rejoué.
4. Le patient peut désactiver les rappels via un lien dans l'email.
5. Les rappels ne sont pas envoyés pour les RDV CANCELLED ou COMPLETED.
