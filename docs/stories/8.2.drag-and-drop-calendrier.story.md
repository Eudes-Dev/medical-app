# Story 8.2: Drag-and-drop dans le calendrier

## Status
Draft

## Story
**As a** praticien,
**I want** déplacer un RDV par glisser-déposer dans la grille,
**so that** je gagne du temps lors des réorganisations d'agenda.

## Acceptance Criteria
1. Un RDV peut être glissé verticalement (changement d'heure) ou horizontalement (changement de jour) dans la vue semaine.
2. Le redimensionnement (drag du bord bas) modifie l'`endTime`.
3. Détection optimiste : si la nouvelle position chevauche un RDV existant, le drop est rejeté avec toast d'erreur.
4. Server Action transactionnelle vérifie le conflit côté serveur.
5. Mise à jour Zustand optimiste + rollback en cas d'échec.
6. Accessibilité : équivalent clavier (focus + flèches + Enter).
