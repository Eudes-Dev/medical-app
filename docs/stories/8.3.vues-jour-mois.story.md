# Story 8.3: Vues Jour et Mois du calendrier

## Status
Draft

## Story
**As a** praticien,
**I want** basculer entre les vues Jour / Semaine / Mois,
**so that** j'adapte ma planification au niveau de détail souhaité.

## Acceptance Criteria
1. Trois vues disponibles : Jour, Semaine (actuelle), Mois.
2. La vue Mois affiche un aperçu compact (nombre de RDV par jour + indicateur de saturation).
3. Le choix de vue est persisté dans Zustand + localStorage.
4. Un clic sur un jour dans la vue Mois bascule en vue Jour à cette date.
5. Navigation clavier (← → ↑ ↓ + Home/End) opérationnelle.

## Tasks / Subtasks
- [ ] Composants `CalendarDayView`, `CalendarMonthView`
- [ ] Extension `useCalendarStore` (view: 'day' | 'week' | 'month')
- [ ] Persistance via middleware `persist` de Zustand
