# Story 9.2: Upload de documents médicaux

## Status
Draft

## Story
**As a** praticien,
**I want** attacher des documents (PDF, images, analyses) à un dossier patient,
**so that** je centralise les pièces médicales.

## Acceptance Criteria
1. Upload multi-fichiers vers Supabase Storage depuis la fiche patient.
2. Types acceptés : PDF, JPG, PNG, DICOM (limite 20 Mo / fichier).
3. Les fichiers sont chiffrés au repos (Supabase native) et signés à l'accès (URL signée 10 min).
4. Tag optionnel par document (Ordonnance, Analyse, Radio, Autre).
5. Suppression réversible 30 jours (soft delete).

## Tasks / Subtasks
- [ ] Bucket Supabase privé `patient-documents`
- [ ] Modèle Prisma `PatientDocument` (storageKey, mimeType, sizeBytes, tag, deletedAt)
- [ ] Server Action upload (validation MIME + scan AV via ClamAV ou option)
- [ ] Composant Dropzone Shadcn
- [ ] Tests permissions
